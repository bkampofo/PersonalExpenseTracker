# PersonalExpenseTracker
Description
Personal Expense Tracker is a Windows Forms application built in C#. It allows users to log and view daily expenses with details like date, category, description, and amount. The app is designed to be simple, user-friendly, and ideal for personal budgeting.


Features
Add expenses with date, category, description, and amount

Input validation for clean and reliable data

Predefined categories for quick selection

View all expenses in a table (DataGridView)

Technologies Used

C#

Windows Forms (.NET Framework)

Regex for input validation

How to Run
Requirements
Windows OS

Visual Studio 2019 or later

.NET Framework 4.7.2 or compatible

Validation Rules
Description: Must be alphabetical, max 100 words


Amount: Must be a positive decimal

Category: Must be selected from the dropdown

Future Improvements
Export expenses to CSV or Excel

Add filtering by date or category

Monthly summary and charts

Persistent storage
